Support

Maintenance Crew for iOS and macOS

Get help

Email privacy@maintenancecrew.app with your question or feedback. Include the app version (Settings → About) and, if relevant, a brief description of what you were doing when the issue occurred.

Common questions

How do I load the sample data?

Go to Settings → Load Sample Data. This populates the app with a demo vessel, schedules, and parts so you can explore features before adding your own boat. You can clear sample data by deleting the demo vessel from the vessel list.

I deleted the app and lost my data. Can I recover it?

All data is stored locally on your device. If you had iCloud Backup enabled, your data may be present in an iCloud Backup that was made before the deletion. Restore from a backup using your device's standard iCloud restore process. Maintenance Crew does not operate any server-side backup; we cannot recover data on your behalf.

Where do I find my Claude API key?

The PDF manual parsing feature requires an API key from Anthropic. Sign in to console.anthropic.com, go to API Keys, and create a new key. Paste it into Settings → Claude API Key in the app.

Your key is stored on your device and is sent only to Anthropic's servers when you initiate a parse. See the Privacy Policy for details. Anthropic may charge for API usage depending on your plan.

The app says it can't get my location.

Location access is used only for Ship's Log entries. If location is unavailable:

  • iOS: Go to Settings → Privacy & Security → Location Services, find Maintenance Crew, and set it to While Using the App.
  • macOS: Go to System Settings → Privacy & Security → Location Services and enable Maintenance Crew. Also make sure Location Services is turned on at the top of that screen.

On macOS, location is derived from Wi-Fi network position and may be less precise than GPS. If you're in an area with limited Wi-Fi networks, the app may not be able to determine your location.

PDF import fails or produces incomplete results.

PDF parsing uses Anthropic's Claude AI. A few things to check:

  • Confirm your API key is entered correctly in Settings.
  • Check that your Anthropic account has available credits or an active API plan at console.anthropic.com.
  • Very large PDFs (>200 pages) may hit API limits; try importing one chapter or section at a time.
  • Scanned PDFs (image-only, no text layer) are not supported — the parser requires selectable text.

If parsing succeeds but specific tasks are missing, you can add them manually from the maintenance schedule screen.
